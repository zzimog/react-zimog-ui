# Field

Use `<Field />` component to compose accesible form fields or grouped inputs, with labels, descriptions, and help texts.

<Demo>
  <Field className="mx-auto max-w-xs">
    <Field.Label>Name</Field.Label>
    <Field.Control />
    <Field.Description>Enter your name here.</Field.Description>
  </Field>
</Demo>

```tsx
<Field>
  <Field.Label>Name</Field.Label>
  <Field.Control />
  <Field.Description>Enter your name here.</Field.Description>
</Field>
```

## Usage

### Label

The `<Field />` component, when used with a `<Field.Label />` component inside, will automatically propagate the label `id` to the `<Field.Control />` component.

A fixed value can be set with the `id` prop on the `<Field />` component.

<Demo>
  <Field id="field-demo-id" className="mx-auto max-w-xs">
    <Field.Label>Name</Field.Label>
    <Field.Control placeholder="Enter your name here" />
  </Field>
</Demo>

```tsx
<Field id="field-demo-id">
  <Field.Label>Name</Field.Label>
  <Field.Control placeholder="Enter your name here" />
</Field>
```

### Description

It is possible to add a description to the field, using the `<Field.Description />` component. The `id` will automatically be set on the `<Field.Control />` component using `aria-describedby` prop.

A fixed value can be set with the `id` prop on the `<Field.Description />` component.

<Demo>
  <Field className="mx-auto max-w-xs">
    <Field.Label>Name</Field.Label>
    <Field.Control />
    <Field.Description
      id="field-demo-description"
      children="Enter your name here."
    />
  </Field>
</Demo>

```tsx
<Field>
  <Field.Label>Name</Field.Label>
  <Field.Control />
  <Field.Description id="field-demo-description">
    Enter your name here.
  </Field.Description>
</Field>
```

### Content

The `<Field.Content />` component is a stack which can be used to visually group items, in both direction using the `direction` prop.

<Demo>
  <Field className="mx-auto max-w-xs">
    <Field.Label>Search</Field.Label>
    <Field.Content direction="horizontal">
      <Field.Control type="search" placeholder="Search" />
      <Button>Search</Button>
    </Field.Content>
  </Field>
</Demo>

```tsx
<Field>
  <Field.Label>Search</Field.Label>
  <Field.Content direction="horizontal">
    <Field.Control type="search" placeholder="Search" />
    <Button>Search</Button>
  </Field.Content>
</Field>
```

### Control

> This component is a work in progress

The `<Field.Control />` component can be used to render the input field.

Without children, it will render a standard `<input />` element.

<Demo>
  <Field className="mx-auto max-w-xs">
    <Field.Label>Name</Field.Label>
    <Field.Control placeholder="Enter your name" />
  </Field>
</Demo>

```tsx
<Field>
  <Field.Label>Name</Field.Label>
  <Field.Control placeholder="Enter your name" />
</Field>
```

If a `ReactElement` is passed as children, it will be used for rendering with props such as `id`, `name` and `aria-describedby`, inherited from the `<Field />` component.

<Demo>
  <Field className="mx-auto max-w-xs">
    <Field.Label>Select an animal</Field.Label>
    <Field.Control placeholder="Choose one">
      <Select>
        <Select.Option value="1">Cat</Select.Option>
        <Select.Option value="2">Dog</Select.Option>
      </Select>
    </Field.Control>
  </Field>
</Demo>

```tsx
<Field>
  <Field.Label>Select an animal</Field.Label>
  <Field.Control placeholder="Choose one">
    <Select>
      <Select.Option value="1">Cat</Select.Option>
      <Select.Option value="2">Dog</Select.Option>
    </Select>
  </Field.Control>
</Field>
```

## Fieldset

Use `<Field.Set />` component to group related form fields together, and the `<Field.Legend />` to set a title for the group.

<Demo>
  <Field.Set className="@container mx-auto max-w-lg">
    <Field.Legend>Address</Field.Legend>
    <Field>
      <Field.Label>Street address</Field.Label>
      <Field.Control />
    </Field>
    <Field>
      <Field.Label>Apt, suite, etc...</Field.Label>
      <Field.Control />
    </Field>
    <div className="flex flex-col gap-3 @sm:flex-row">
      <Field className="basis-2/3">
        <Field.Label>City</Field.Label>
        <Field.Control />
      </Field>
      <Field className="basis-1/3">
        <Field.Label>ZIP code</Field.Label>
        <Field.Control />
      </Field>
    </div>
  </Field.Set>
</Demo>

```tsx
<Field.Set className="@container">
  <Field.Legend>Address</Field.Legend>
  <Field>
    <Field.Label>Street address</Field.Label>
    <Field.Control />
  </Field>
  <Field>
    <Field.Label>Apt, suite, etc...</Field.Label>
    <Field.Control />
  </Field>
  <div className="flex flex-col gap-3 @sm:flex-row">
    <Field className="basis-2/3">
      <Field.Label>City</Field.Label>
      <Field.Control />
    </Field>
    <Field className="basis-1/3">
      <Field.Label>ZIP code</Field.Label>
      <Field.Control />
    </Field>
  </div>
</Field.Set>
```
